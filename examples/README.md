# Examples

Small, self-contained programs that show how to use vsdx-go.

| Example | What it does |
|---------|--------------|
| [`inspect`](inspect) | Open a `.vsdx` file and print its pages and shape text |

Run one with:

```bash
go run ./examples/inspect path/to/diagram.vsdx
```

For API-level snippets (open, edit + save, render to SVG), see the runnable
examples on [pkg.go.dev](https://pkg.go.dev/github.com/wijnberg-net/vsdx-go/vsdx#pkg-examples).
